rx = rotação no eixo x <br>
ry = rotação no eixo y <br>
rz = rotação no eixo z <br>
tm = transformação <br>
nd = matriz que não modifica as matrizes de rotação <br>
nd2 = matriz que não modifica a matriz de transformação <br>

por padrão: cube.render(rx,ry,rz,nd2) <br>
